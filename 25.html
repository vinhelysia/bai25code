Sơ đồ Tương tác Hàm Python

Hãy bấm vào các lệnh có nét đứt trong đoạn code dưới đây để xem giải thích (kèm mũi tên chỉ dẫn).

# Chương trình xử lý văn bản tổng hợp Bài 25
van_ban = input('Nhập một câu: ')

if 'lỗi' in van_ban:
    vt = van_ban.find('lỗi')
    print('Cảnh báo: Tìm thấy chữ lỗi ở vị trí ' + str(vt))

danh_sach_tu = van_ban.split()
so_luong_tu = len(danh_sach_tu)

tu_hop_le = []
for tu in danh_sach_tu: # Duyệt từng từ trong danh sách
    tu_hop_le.append(tu)

cau_hoan_chinh = ' '.join(tu_hop_le)
print('Kết quả: ' + cau_hoan_chinh)

Bấm vào một hàm/lệnh ở đoạn code phía trên để hiển thị hộp giải thích chi tiết.
